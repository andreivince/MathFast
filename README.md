# Aura Math

Aura Math is a local-first mental math PWA for a single daily 10-minute session.

## Scope

- Daily mission with a fixed 10-minute flow
- Timed arithmetic drills
- Weak facts tracked from wrong and slow-correct answers
- Pressure mode
- Flash anzan
- Local stats stored in the browser
- No login, backend, multiplayer, social layer, subscription, or AI tutor

## Run

Serve the folder with any static server:

```sh
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
